# ✨ Stylish Task Manager (CRUD Web App)

A simple and elegant **Task Manager Web Application** built using **HTML, CSS, and JavaScript**. This app allows users to manage their daily tasks with full **CRUD functionality** (Create, Read, Update, Delete) — all in a clean, modern interface.

---

## 🚀 Features

* ➕ **Add Tasks** – Create new tasks easily
* 📖 **View Tasks** – See all your tasks in a structured list
* ✏️ **Edit Tasks** – Update task content anytime
* ✔️ **Mark as Completed** – Toggle task completion
* ❌ **Delete Tasks** – Remove tasks you no longer need
* 💾 **Persistent Storage** – Tasks are saved in browser using `localStorage`

---

## 🖥️ Technologies Used

* HTML5
* CSS3 (Modern UI with gradients & styling)
* JavaScript (Vanilla JS, no frameworks)

```
Double-click OR Right-click → Open with browser
```

No installation or dependencies required 🎉

---

## 🧠 How It Works

* Tasks are stored as an array of objects:

```
{ text: "Task name", done: false }
```

* Data is saved in:

```
localStorage
```

* Every action (add/edit/delete/toggle) updates both:

  * UI (DOM)
  * Storage (localStorage)
